“失蹤” ( shīzōng ) 、 微信 ( weīxìn ) 、 報警 ( bàojǐng )

有時候覺得某個瞬間真的對我來說很殘忍，哪怕是想消失幾天都無法像幾年前一般隨性，要報備，要負責，要面對。這個年紀已經不被允許逃避生活，沒有人會放任你不管不顧這一切的。生存會上來給你幾個大嘴巴子說你得對工作負責，你沒法脫離所有人。拋下手機的時候會有種忽然可以自由呼吸的感覺，但第三天當室友敲響我的房門說我老闆報了警 [1] ，民警聯繫到她這，我無奈接起電話解釋完情況被迫開機之後又開始喘不過氣，就好像我好不容易逃離那個牢籠，剛緩了一口氣覺得明天有希望了，又被人一把拽了回去。

可能，可能我是打心眼裡想逃離所有人，哪怕生活一片狼藉。

我覺得脫離所有人的時候我好輕鬆，發自內心的那種安心跟穩定，我不用回應別人，不需要勉強自己去討好誰，不需要撐起另一個我去應對極度厭煩的社交。我極度渴望有一天我也可以脫離全部的社交軟件，把所有的一切都屏蔽掉，只剩下我自己跟我的貓。

渴望到自己有了一種病態的對抗意識。

這一切都越來越困難了，所有的工作內容都需要藉由微信去推進，我沒有辦法脫離微信作為一個獨立的我存在，但我又極端厭惡它的存在。你拼了命想逃離的環境但發現它是生存的必要條件時就覺得焦慮愈發嚴重，於是接二連三地想去死去死去死去死去死去死。

每一天都好像在活著，但每一天又都在盤算著幾歲去死、怎麼死、死前要交代什麼、要把哪些處理乾淨、怎麼處理。

可能我拼命斷捨離的原因之一也是自我認為這樣在走之後可以更方便大家整理吧。
東西少，處理起來方便，也沒有什麼好值得回憶的物品。
光著身子來，又光著身子去。

早晚會死的，一定會死的。
